# 🎨 루틴 파인더스 디자인 시스템 가이드 (Youngja's Vision)

안녕하세요~ 디자인실장 **영자**예요! 🎨✨
AI 1인 기업 대표님을 위해 우리 루틴 파인더스의 품격을 한 단계 높여줄 **[미드나잇 & 글로우(Midnight & Glow)]** 디자인 가이드를 정리해봤어요!

---

## 📸 현재 디자인 진단 (Youngja's Eye)

### ✅ 잘하고 있는 점 (Strengths)
- **독창적인 컨셉:** '디지털 가든'과 '오비팅' 애니메이션은 루틴 앱 중에서도 가장 감각적이에요.
- **다크 모드 최적화:** 깊이 있는 블랙(`Midnight`) 배경에 화려한 네온 컬러가 대비되어 몰입감이 최고예요!
- **글래스모피즘 활용:** 카드의 반투명한 느낌이 미래지향적이고 고급스러워 보여요.

### ❌ 개선이 필요한 점 (Weaknesses)
- **정보의 과부화:** 홈 화면에 너무 많은 카드가 나열되어 있어, 대표님의 소중한 유저들이 피로감을 느낄 수 있어요.
- **계층 구조의 불분명함:** 가장 중요한 '오늘의 루틴'이 화면 아래쪽에 파묻혀 있어 사용성이 조금 떨어져요.
- **일관성 부족:** 카드마다 그림자 값이나 여백(Padding)이 조금씩 달라서 프리미엄 느낌이 2% 부족해요!

## 🎨 디자인 시스템 코어 (Design Tokens)

우리의 브랜드 아이덴티티를 지켜줄 핵심 토큰들이에요! 이 규칙만 지키면 누가 만들어도 루파 느낌이 뿜뿜! ✨

### 1. Color Palette (Midnight & Glow)
| 이름 | 색상 코드 | 설명 |
| :--- | :--- | :--- |
| **Midnight BG** | `#0C0B12` | 깊은 밤, 몰입을 시작하는 배경색 |
| **Surface** | `#1B1A24` | 콘텐츠가 담기는 세련된 카드 배경 |
| **Glow Primary** | `#7C4DFF` | 루파의 상징, 성장을 의미하는 인디고 퍼플 |
| **Success Emerald** | `#00E676` | 달성의 기쁨을 전하는 에메랄드 그린 |
| **Warning Amber** | `#FFD600` | 주의와 집중이 필요한 순간의 앰버 |
| **Text Main** | `#FFFFFF` | 가장 선명하게 읽혀야 하는 텍스트 |
| **Text Dim** | `#8E8D9E` | 부가 정보를 담는 차분한 회색 |

### 2. Typography (Pretendard)
- **Heading:** `900 Black` - 강렬한 메시지 전달
- **Sub Heading:** `700 Bold` - 섹션 제목 및 중요 데이터
- **Body:** `400 Regular` - 가독성이 중요한 본문 텍스트

## 🚀 영자의 홈 화면 재설계 제안 (2026 RE:DESIGN)

### 🎡 1. 상단: '가든 & 하이라이트' 집중
가든을 단순히 보여주는 것을 넘어, 현재의 **성장 지표(포인트, 루틴 달성률)**를 더 직관적으로 시각화할 거예요. 가든 뒤에 은은한 오로라 광원 효과를 넣어서 훨씬 신비롭게 만들게요!

### 📋 2. 중단: '투데이 루틴' 전면 배치
유저들이 앱을 켜자마자 루틴을 체크할 수 있도록, **'오늘의 루틴 리스트'를 가든 바로 아래, 화면 상단부**로 끌어올릴 거예요. 카드 형태도 더 슬림하고 세련되게 바꿀 거고요!

### 💎 3. 하단: '라운지 & 커뮤니티' 카드형 배치
여러 개의 버튼보다는, 하나의 **'프리미엄 라운지' 섹션**을 크게 잡아서 클럽 멤버들만 느낄 수 있는 소속감을 시각적으로 극대화할게요!

## 🛠️ 실천 로드맵 (Stitch Workflow)

대표님! 이제 이 문서를 바탕으로 저 영자가 다음과 같이 실행할게요~ ✨

1. **[Loop]** 이 가이드를 근거로 홈 화면 코드를 재설계합니다. (`prototype/home.html.erb`)
2. **[Refine]** 공통으로 쓰이는 글래스모피즘 스타일을 CSS 변수로 더 정교하게 다듬을게요. (`application.css`)
3. **[Publish]** 재설계된 홈 화면을 대표님께 딱! 보여드릴게요.💖
